# Endless: Reward Level Rate Haste Mod

This is a very simple mod that replaces every how many levels the reward screen appears in Endless mode.

It also serves as a simple introductory example to MonoMod `ILHooks`.

Check out [RewardEndlessLevelRateMod.cs](./HasteEndlessRewardLevelRateMod/RewardEndlessLevelRateMod.cs) to get started!

For more resources, refer to [our wiki](https://github.com/HasteModding/HasteModding/wiki) or [#haste-mod-development-chat](https://discord.gg/hastebrokenworlds) in Haste's official Discord server.
